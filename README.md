# Reswipe App (reswipe)

A Quasar Project

## Install the dependencies
```bash
yarn
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
yarn quasar dev
```


### Lint the files
```bash
yarn lint
```


### Format the files
```bash
yarn format
```


### Build the app for production
```bash
yarn quasar build
```
